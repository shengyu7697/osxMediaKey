# osxMediaKey
Simple OS X application to trigger the media key.  

Support media key:  
- Brightness down
- Brightness up
- Rewind
- Play
- Fast
- Mute
- Volume down
- Volume up

# Usage
```
make
./osxMediaKey [brightnessdown|brightnessup|rewind|play|fast|mute|voldown|volup]
```
